# Shubh Law Firm Website

A professional and modern website for Shubh Law Firm built with Next.js 14, JavaScript, and Tailwind CSS.

## 🚀 Features

- **Modern Design**: Clean, professional layout with the law firm's brand colors
- **Responsive**: Mobile-friendly design that works on all devices
- **Fast Performance**: Built with Next.js 14 for optimal speed and SEO
- **Interactive Elements**: Smooth animations and hover effects
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Proper meta tags and structured content

## 🎨 Design System

### Color Palette
- **Primary Brown**: #2d1810 (Deep brown/black)
- **Primary Orange**: #ff6b35 (Warm orange)
- **Accent Blue**: #64748b (Blue-gray)
- **Accent Red**: #dc2626 (Subtle red tones)
- **Neutral White**: #ffffff (Clean white background)

### Typography
- **Font**: Inter (Google Fonts)
- **Clean, professional typography hierarchy**

## 🛠️ Technologies Used

- **Next.js 14.2.30** - React framework with App Router
- **JavaScript** - No TypeScript as requested
- **Tailwind CSS** - Utility-first CSS framework
- **React Hooks** - State management and effects
- **CSS Animations** - Custom keyframes and transitions

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd shubhlawfirm
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🔧 Customization

### Colors
Update the color palette in `tailwind.config.js` and `src/app/globals.css`

### Content
- Update text content in respective page components
- Replace placeholder images in `public/images/`
- Modify contact information in components

### Styling
- Customize animations in `tailwind.config.js`
- Update global styles in `src/app/globals.css`
- Modify component-specific styles

## 📱 Responsive Design

The website is fully responsive with breakpoints:
- **Mobile**: < 768px
